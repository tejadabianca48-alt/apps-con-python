# **TRABAJO INTEGRADOR** 
Diseño de dos programas interactivos en Python y ejecutables desde la terminal.
---

## Tateti

Un juego clásico de la infancia llevado a un programa, el cual es contra la computadora.

- Visualización del tablero en la terminal
- Humanos vs Computadora (que gane el mejor)
- Utiliza la librería random
---

## Generador de Tickets
Este es un programa generador y lector de Tickets, de esta forma a partir de un código random se podra hallar un antiguo ticket o crear uno en relación al cotexto que el ususario desee.

Librerías empleadas
-json
-random
-Pyckle
