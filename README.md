# **TRABAJO INTEGRADOR** 
Diseño de dos programas interactivos en python y ejecutables desde la terminal.
---

## Tateti

Un juego clasico de la infanci llevado a un programa, el cual es contra la computadora.

- Visualización del tablero en la terminal
- Humanos vs Computadora (que gane el mejor)
- Utiliza la libreria random
---

## Generador de Tickets
Este es un programa generador y lector de Tickets, de esta forma a parti de un codigo random se podra hallar un atiguo ticket o crear uno en relación al cotexto que el ususario desee.
